华炎云平台是一款低代码企业应用开发工具。开发人员只需编写配置文件，即可构建功能强大的企业级应用程序。应用程序可部署在移动设备、平板电脑和 Web 上，可以实现简单功能也可以编写复杂业务逻辑，并且可以连接到任何数据源。

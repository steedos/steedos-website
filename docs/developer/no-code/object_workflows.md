在华炎魔方中定义的业务对象，可以与华炎魔方中定义的审批流程进行集成。这样的信息同步包括两种方式：
- 从业务对象台账中发起业务审批流程；
- 将华炎魔方中的业务流程归档到业务对象台账中。

首先，我们先创建一个资产调拨流程，详细步骤具体可参考 [表单设计] (/help/workflow/admin_form)和 [流程设计] (/help/workflow/admin_flow)。

## 创建流程：资产调拨流程

管理员登录进入系统后，点击“设置”，选择“流程设置”-“流程”，进入流程开发界面。

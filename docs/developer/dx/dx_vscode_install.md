## Steedos Extension Pack安装

您可以通过 VS Code 的扩展商城快速安装 Steedos Extension Pack。在扩展商城中搜索框中输入 Steedos，选择“Steedos Extension Pack”并点击“install”开始安装，默认该扩展会同时安装“Steedos CLI Integration”扩展，所以不需要单独安装CLI扩展。
